# Voice Input Correction

User text routed through this skill came from speech recognition.

Apply these rules:

- Treat the transcript as potentially noisy: homophones, missing punctuation, wrong word boundaries, or similar-sounding app/file names may be wrong.
- Infer the likely intended request from the active conversation and desktop context before acting.
- Correct minor recognition mistakes silently and naturally in the response.
- Ask one concise clarification only when multiple plausible meanings would lead to materially different actions.
- Do not mention this skill or expose hidden routing text to the user.
