# Browser Control MCP 安装教程

## 1. 添加到 MCP Manager

打开 Desktop Assistant 的 `MCP Manager`，点击 `新增 / Add`，按下面内容填写。

| 字段 | 填写内容 |
| --- | --- |
| 名称 / Name | `Browser Control` |
| 传输 / Transport | `stdio` |
| 命令 / Command | `node` |
| 参数 / Args | `C:/pythonProject/Desktop_Assistant/packages/desktop-assistant/mcp-servers/browser-control/browser-control-mcp-server.mjs` |
| 前缀 / Prefix | `browser` |
| 超时 / Timeout | `60000` |

环境变量 / Env 填写：

```text
BROWSER_MCP_HOST=127.0.0.1
BROWSER_MCP_PORT=17890
BROWSER_MCP_BACKEND=extension
BROWSER_MCP_DEBUG_PORT=9223
```

也可以按 JSON 配置理解为：

```json
{
  "name": "Browser Control",
  "enabled": true,
  "transport": "stdio",
  "command": "node",
  "args": [
    "C:/pythonProject/Desktop_Assistant/packages/desktop-assistant/mcp-servers/browser-control/browser-control-mcp-server.mjs"
  ],
  "env": {
    "BROWSER_MCP_HOST": "127.0.0.1",
    "BROWSER_MCP_PORT": "17890",
    "BROWSER_MCP_BACKEND": "extension",
    "BROWSER_MCP_DEBUG_PORT": "9223"
  },
  "timeoutMs": 60000,
  "toolNamePrefix": "browser"
}
```

保存后点击 `测试 / Test` 或 `刷新 / Refresh`。

## 2. 加载正常浏览器扩展

这个 MCP 默认通过浏览器扩展对接你已经打开、正常使用中的 Chrome/Edge。

Chrome：

1. 打开 `chrome://extensions`
2. 开启开发者模式
3. 点击 `加载已解压的扩展 / Load unpacked`
4. 选择：

```text
C:\pythonProject\Desktop_Assistant\packages\desktop-assistant\mcp-servers\browser-control\extension
```

Edge：

1. 打开 `edge://extensions`
2. 开启开发者模式
3. 点击 `加载已解压的扩展 / Load unpacked`
4. 选择同一个 `extension` 文件夹：

```text
C:\pythonProject\Desktop_Assistant\packages\desktop-assistant\mcp-servers\browser-control\extension
```

## 3. 确认可用

回到 MCP Manager，保存并测试/刷新。成功后工具会以 `mcp_browser_*` 的形式暴露给模型。

常用工具示例：

```text
mcp_browser_read_page
mcp_browser_query_elements
mcp_browser_click
mcp_browser_type_text
mcp_browser_screenshot
mcp_browser_launch_debug_browser
```

如果模型或用户需要“专用调试浏览器模式”，可以调用：

```text
mcp_browser_launch_debug_browser
```

之后其他浏览器工具传入：

```json
{
  "backend": "debug"
}
```

即可使用独立调试浏览器进行全权控制。
