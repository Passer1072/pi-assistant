# 更新日志

本文件记录 Pi 桌面助手（packages/desktop-assistant）的重要更新。
遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/) 风格。

## [未发布] - 2026-06-20

### 新增

- **备忘录（待办 + 提醒）**：全新的待办事项系统。
  - 支持截止日期、独立提醒时间、优先级、标签、子任务清单、重复提醒（每天/每周/每月）、置顶、颜色标记。
  - 逾期 / 今天 / 即将 / 无日期 / 已完成分组，支持搜索、排序与快速添加。
  - 数据持久化于 `userData/memos/memos.json`（原子写），跨重启保留。
  - **三路提醒**：应用内横幅 + Toast、Windows 系统通知（点击聚焦窗口并打开备忘录）、桌宠气泡。
  - **错过提醒补播**：应用关闭期间到点的提醒会在下次启动时补发。
  - **AI 深度联动**：新增 `memo_create / memo_list / memo_search / memo_update / memo_complete / memo_set_reminder / memo_delete` 工具，可用自然语言创建、查询、完成待办；提醒到点且会话空闲时，助手会主动在聊天中提醒。
  - 入口位于左侧抽屉「新建并行会话」下方，带今日 / 逾期数角标。

- **独立首页**：带时段问候、标语、今日提醒、快捷入口、示例问题与输入框的落地页；入口位于抽屉最上方，启动默认进入。
  - **文本发送**（按钮 / 回车）= 新建会话并进入对话页。
  - **语音 / 唤醒**（在首页）= 每次新建一个全新会话，并在首页空白区**内联**显示，不跳转。
  - 首页输入框支持拖拽 / 粘贴文件附件预览。

- **首页沉浸式内联对话**：语音对话在首页空白区原位显示，顶部以虚化遮罩自然融入上方快捷卡片（无分界线、无大边框）；点击对话区可丝滑展开为完整会话页。
  - 输入框复用语音状态光晕（聆听绿 / 识别橙 / 错误红 / 聚焦蓝）。
  - 用户气泡向上浮现淡入，模型流式回答逐字淡入。
  - 空闲 5 秒（无新语音输入或应用内操作）自动清空，回到干净首页。

- **聊天页提醒条**：聊天页顶部显示今日 / 逾期待办，点击进入备忘录页，可关闭。

### 变更

- 页面路由由横向滑轨（page-stack）改为**覆盖层模型**，支持首页 / 对话 / 备忘录 / 设置之间的平滑单段过渡。
- 左侧抽屉新增「首页」「备忘录」入口，布局由 grid 改为 flex。

### 测试

- 新增 `memo-repository`、`memo-reminder-scheduler`、`memo-tools` 单元测试。
- 全量 335 项测试通过；`npm run build` 通过。
