# 更新日志

本文件记录 Pi 桌面助手（packages/desktop-assistant）的重要更新。
遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/) 风格。

## [未发布] - 2026-06-28

### 新增

- **灵动窗 (Dynamic Window)**：每个会话右侧浮出多 facet 浮窗，实时可视化模型正在操作的对象。
  - `文件树` facet：模型读写的文件以目录树展示，可展开预览。
  - `网页快照` facet：浏览器工具调用后自动截图并嵌入面板。
  - `命令终端` facet：Shell 工具输出流式滚动。
  - `流程图` facet：自动化执行路径实时高亮（已走边变绿），复用 `FlowGraphPreview`。
  - 面板可拖拽定位、折叠 / 展开、固定 / 跟随，历史会话可重建快照。

- **更多应用平台 (More Apps)**：外部本地 Web 应用接入，各应用独立 Electron 窗口 + 子进程桥管理。
  - 已接入：**email-manager**（邮件 + AI 摘要）、**E-Book 阅读器**（阅读 + AI 批注）。
  - 新增应用只需在 `external-app-registry` 注册配置项，主进程自动管理生命周期。
  - AI 可通过 `app_bridge_*` 工具与应用内 UI 交互。

- **首页 AI 欢迎语**：首页问候改由 DeepSeek Flash 动态生成，包含今日待办数、自动化数量、天气简报、邮件摘要。
  - 30 min 签名成本闸门防止频繁生成；HELLO 占位打字机动画保证加载感知。

- **首页天气小组件**：问候区右上角天气卡片（WeatherAPI + lucide 图标 + 2 h 缓存）；设置页填入 `WeatherAPI Key` 即可显示。

- **自动化流程图气泡**：自动化会话置顶显示当前流程进度气泡；已走路径实时变绿；按固定 id 原地更新 timeline 项，历史可重建。

- **实时流程化 (Live Flow)**：普通会话内模型自驱实时设计并执行流程。
  - 右下角可拖拽折叠浮窗实时渲染流程图。
  - 三个专用工具（`flow_init / flow_step / flow_end`）+ `LiveFlowSession` 独立会话管理。
  - `flow_step` 回传下一步说明防止模型跑偏；通过 `profile.extraTools` 按会话按需注入。

- **出错自我总结**：模型遇工具报错后自动把流程复盘记成备忘录。
  - 每轮至多一条复盘 memo，排除用户主动取消、简单输入错误等非模型失误。
  - 新增 `session_info` 工具供模型感知当前会话信息。

### 优化与修复

- **长会话 OOM 修复**：主进程 V8 堆上限提升；浏览器工具完成后压缩并回写快照，不再驻留原始大块；消息展示滑动窗口 + 日志行数上限双重保险。
- **Token 节省 / 冻结式压缩**：针对 DeepSeek 前缀缓存被快照逐出击穿导致 Token 暴涨的根因修复；`frozenForms` 冻结式上下文压缩保持消息前缀稳定。
- **网易云 MCP**：修复 CDP 重连后工具调用超时问题。
- **浏览器工具**：新增 `browser_scroll` 与 `browser_wait` 工具；改进错误信息格式。

## [1.1.0] - 2026-06-20

### 新增

- **备忘录（待办 + 提醒）**：全新的待办事项系统。
  - 支持截止日期、独立提醒时间、优先级、标签、子任务清单、重复提醒（每天/每周/每月）、置顶、颜色标记。
  - 逾期 / 今天 / 即将 / 无日期 / 已完成分组，支持搜索、排序与快速添加。
  - 数据持久化于 `userData/memos/memos.json`（原子写），跨重启保留。
  - **三路提醒**：应用内横幅 + Toast、Windows 系统通知（点击聚焦窗口并打开备忘录）、桌宠气泡。
  - **错过提醒补播**：应用关闭期间到点的提醒会在下次启动时补发。
  - **AI 深度联动**：新增 `memo_create / memo_list / memo_search / memo_update / memo_complete / memo_set_reminder / memo_delete` 工具，可用自然语言创建、查询、完成待办；提醒到点且会话空闲时，助手会主动在聊天中提醒。
  - 入口位于左侧抽屉「新建并行会话」下方，带今日 / 逾期数角标。

- **独立首页**：带时段问候、标语、今日提醒、快捷入口、示例问题与输入框的落地页；入口位于抽屉最上方，启动默认进入。
  - **文本发送**（按钮 / 回车）= 新建会话并进入对话页。
  - **语音 / 唤醒**（在首页）= 每次新建一个全新会话，并在首页空白区**内联**显示，不跳转。
  - 首页输入框支持拖拽 / 粘贴文件附件预览。

- **首页沉浸式内联对话**：语音对话在首页空白区原位显示，顶部以虚化遮罩自然融入上方快捷卡片（无分界线、无大边框）；点击对话区可丝滑展开为完整会话页。
  - 输入框复用语音状态光晕（聆听绿 / 识别橙 / 错误红 / 聚焦蓝）。
  - 用户气泡向上浮现淡入，模型流式回答逐字淡入。
  - 空闲 5 秒（无新语音输入或应用内操作）自动清空，回到干净首页。

- **聊天页提醒条**：聊天页顶部显示今日 / 逾期待办，点击进入备忘录页，可关闭。

### 变更

- 页面路由由横向滑轨（page-stack）改为**覆盖层模型**，支持首页 / 对话 / 备忘录 / 设置之间的平滑单段过渡。
- 左侧抽屉新增「首页」「备忘录」入口，布局由 grid 改为 flex。

### 测试

- 新增 `memo-repository`、`memo-reminder-scheduler`、`memo-tools` 单元测试。
- 全量 335 项测试通过；`npm run build` 通过。
